Julia language support.

  • Syntax highlight
  • Language injection
  • Run configuration and code execution
  • Live templates
  • File templates
  • Completions
  • Jump to definition

  • Renaming
  • Doc string generation
  • Try Evaluate
  • Bread crumbs
